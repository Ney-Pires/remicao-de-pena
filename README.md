⚖️ Calculadora de Remição de Pena (Projeto UNIVESP)

Este projeto foi desenvolvido como Projeto Integrador I **UNIVESP** e tem como objetivo automatizar o cálculo da remissão de pena de detentos, facilitando o controle jurídico e administrativo com base na legislação brasileira.

 📖 Contexto e Previsão Legal
O projeto baseia-se no **Artigo 126 da Lei de Execução Penal (LEP)**, que estabelece que o condenado pode remir parte do tempo de execução da pena por meio de:

Trabalho:** 1 dia de pena a menos para cada 3 dias trabalhados.
* **Estudo:** 1 dia de pena a menos para cada 12 horas de frequência escolar (divididas em, no mínimo, 3 dias).
* **Leitura:** Remição por leitura de livros conforme recomendações do CNJ.

> "A cada três dias de trabalho ou doze horas de estudo permite-se a remissão de um dia de pena." — **Art. 126, LEP.**

## 🛠️ Tecnologias Utilizadas
* Python (Lógica de cálculo)
* Git/GitHub (Versionamento)

## 🚀 Como Executar o Projeto
Se você deseja testar o projeto localmente, siga os passos abaixo:

1. Clone o repositório:
   `git clone https://github.com/Ney-Pires/remicao-de-pena.git`
2. Entre na pasta:
   `cd remicao-de-pena`
3. Instale as dependências:
   `pip install -r requirements.txt`
4. Execute o programa principal:
   `python remicao/seu_arquivo_principal.py`
